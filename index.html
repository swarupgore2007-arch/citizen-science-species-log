Rarity Index Explained

The rarity index is calculated using:

R = Actual Sightings / Expected Annual Frequency
  • R < 0.5 Rare / Unexpected
  • 0.5 <= R <= 1.5 As Expected
  • R > 1.5 Common
  • < 3 sightings Insufficient Data
